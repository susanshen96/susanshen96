Snap! Reference Manual: Written by Brian Harvey and Jens Mönig, this PDF manual offers a walkthrough of the various features in Snap!. Provided in this Snap! manual are explanations of the functions of the provided blocks, along with tutorials that allow the user to test these functions of Snap!.


CS10: The Beauty and Joy of Computing: CS10 is a course offered in UC Berkeley that introduces students to the basics of computer programming through Snap!. Taught in the past by professors Dan Garcia and Gerald Friedland, CS10 provides its teaching materials online. Here, users can browse labs and lectures, all of which are useful tools in helping understand the various facets of Snap!.
