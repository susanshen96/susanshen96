Stuck? These resources will help you in a

Snap! Reference Manual: This manual is both a reference and a tutorial, especially with respect to Snap! features that aren't also in Scratch: lists as a first class data type, procedures (blocks and scrpts) as first class data, object oriented programming, and continuations.


Help Screens: Right-click on any block to see an explanation of what that block does, with examples of how to use it.


User Forum: Go to https://scratch.mit.edu/discuss/topic/4455 to ask for programming help.


Example projects: While running Snap!, go to the file menu, choose "Open," and then click the "Examples" button. You will then see a selection of example projects that demonstrate various features of the language.


CS10: The Beauty and Joy of Computing: CS10 is a course offered at UC Berkeley that introduces students to the basics of computer programming through Snap!. we provide online labs and lectures, many of which are useful tools in helping understand the various facets of Snap!.
